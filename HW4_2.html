프로젝트 TYP (Tag Your Photo)

1. 프로젝트 개요

이때까지 경험해본 사진 정리 프로그램은 정렬 기준이 이름/날짜/장소에 집중되어 있다. 비록 각 사진마다 태그를 달 수는 있지만, 태그 입력/정렬/삭제등에 UX가 집중되어 있지 않다. 사용자들이 자기가 원하는 태그를 쉽게 달고 검색 및 수정할 수 있는, Tag 집중 프로그램을 만들 것이다.

2. User Analysis / Research

A. Target User

사진들을 전문적으로 찍기보다는, 평소에 사진을 따로 정리하지 않는 이용자들이 더 쉽고 편하게 정리하고 빠르게 원하는 사진을 찾을 수 있도록 도와주는 프로그램이다. 사진을 많이 찍고 주고받는 10대부터 30대가 타겟이고 그들의 스킬레벨은 Novice다.

B. Devices and Platforms

해당 프로그램은 유저들이 사진을 컴퓨터에 가져와서 정렬하는 프로그램이다. Windows와 Mac와 같은 운영체제를 갖고 있는 PC에서만 사용 가능하다.

3. Requirements

A. Functional

  • 날짜/장소/태그 기준으로 사진들을 분류
  • 날짜/장소/태그 기준으로 사진들을 검색
  • 손쉽게 태그 추가 및 수정
    • 태그 추가를 할때 인공지능이 사진을 분석해서 추천 태그를 적용
    • 빠르고 단순한 동작으로 추가 수정이 가능

      B. UI

  • Functional
    • 태그 추가 및 수정을 할때 최소한의 거리와 클릭 횟수를 요구
  • Non-Functional
    • 태그의 내용이나 정보를 쉽게 볼 수 있도록 분류하여 시각화
    • 불필요한 문자 제외 (Memory 최소화)

4. Scenario

A. 처음 사용자

  1. 프로그램 실행
  2. 사진 검색 범위 설정
  3. 사진 검색 및 자동 분류
  4. 태그 분류창으로 이동

B. 태그 일반 설정

  1. 태그 분류창으로 이동
  2. 원하는 사진 선택
  3. 설정하고 싶은 태그를 클릭

C. 태그 빠른 설정

  1. 태그 분류창으로 이동
  2. 설정하고 싶은 태그를 사진에 드래그

D. 사진 태그 검색

  1. 사진 검색창으로 이동
  2. 원하는 태그를 클릭

5. Task Model

6. Interface

A. 태그 분류창

사진 검색 혹은 태그 분류 선택

  • 다양한 사진들에 작업해야 하기 때문에 공간을 크게 배정
  • 많은 태그 정보들을 보여주기 위해 공간을 크게 배정

B. 사전 검색창

  • 태그를 통한 검색을 유도하기 위해 왼쪽에 배치
  • 사진과 그 정보가 중요하기 때문에 공간을 크게 배치
